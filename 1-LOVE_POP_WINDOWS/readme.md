# 爱心祝福弹窗 ❤️

一个基于 Python + Tkinter 开发的**爱心轨迹悬浮祝福弹窗**，会按照数学爱心曲线依次弹出温馨文案窗口，搭配多彩柔和背景，氛围感拉满～

## ✨ 功能特点

1. **爱心轨迹动画**：通过数学公式生成标准爱心曲线，弹窗沿爱心轨迹依次弹出
2. **多彩悬浮窗口**：马卡龙色系柔和背景，半透明+边框美化，视觉轻盈温馨
3. **随机暖心文案**：内置多条治愈系祝福，随机展示不重复
4. **全局置顶显示**：弹窗始终在桌面最上层，视觉效果拉满
5. **一键退出**：按下 `ESC` 键即可关闭所有弹窗，操作简单
6. **纯原生库开发**：无需安装第三方依赖，开箱即用

## 🖥️ 运行环境

- Python 3.x（推荐 3.7 及以上版本）
- 内置依赖：`tkinter`（Python 自带，无需额外安装）、`random`、`math`

## 🚀 使用方法

### 1. 保存代码

将代码保存为 `heart_popup.py` 文件

### 2. 运行程序

直接执行代码：

```bash
python heart_popup.py
```

### 3. 效果展示

程序启动后，会先沿**爱心曲线**弹出弹窗，曲线播放完毕后，会在屏幕随机位置继续弹出，直至完成所有弹窗。

## 🎨 自定义修改

你可以轻松修改文案、颜色、弹窗数量等参数，打造专属效果：

### 1. 修改祝福文案

找到代码中 `messages` 列表，直接增删改字符串即可：

```python
messages = [
    "保持好心情",
    "今天过得开心嘛",
    "你的自定义文案",  # 在这里添加/修改
]
```

### 2. 修改弹窗颜色

找到 `colors` 列表，替换十六进制颜色码：

```python
# 马卡龙色系，可自行替换
colors = ["#ffd1dc", "#cdeffd", "#d9fdd3", "#ffe7c7", "#efe0ff", "#fff3b0"]
```

### 3. 调整弹窗数量/速度

```python
popups = PopupManager(
    root,
    messages,
    colors,
    count_heart=180,    # 爱心轨迹弹窗数量
    count_random=160,   # 随机弹窗数量
    interval_ms=25,     # 弹出间隔（越小速度越快）
    size=(240, 70),     # 窗口尺寸(宽,高)
)
```

## ⌨️ 退出方式

按下键盘 **`ESC` 键**，一键关闭所有弹窗，结束程序。

## 📝 核心原理

- 爱心曲线：使用**笛卡尔爱心方程**计算坐标，让弹窗完美沿爱心轨迹排列
- GUI 实现：基于 Python 内置 `tkinter` 实现无标题栏悬浮窗口
- 动画效果：通过定时器逐帧创建弹窗，实现流畅的弹出动画
